# Replate

Replate is a web application that helps buffets track food waste, schedule food donations, and measure their sustainability impact.

The platform allows restaurants to log daily waste, connect with nearby shelters or food banks, and view a sustainability score based on waste reduction and food redistribution.

---

## Features

- Multi-step onboarding for restaurant setup  
- Login and reset password pages  
- Dashboard overview of restaurant data  
- Log daily food waste entries  
- Interactive donation partner map
- Sustainability score visualization  
- LocalStorage-based authentication

---

## Tech Stack

This project is built with:

- **Next.js**
- **React**
- **Tailwind CSS**
- **Leaflet**
- **TypeScript**
- **LocalStorage**

## Getting Started

First install dependencies:
```bash
npm install
```

Then run the development server:
```bash
npm run dev
```

Open the app in your browser:
```bash
http://localhost:3000
```
